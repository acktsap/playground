## Imperative Programming vs Declarative Programming

명령형 프로그래밍은 어떻게(how) 할 것인지에 집중하는 것이고 선언형 프로그래밍은 무엇을(what) 할 건지에 집중하는 것이다. 결국은 명령형으로 내부적으로 돌아가기 때문에 선언형이 제대로 작동하기 위해서는 명령형으로 어떻게 할건지가 충분히 추상화가 되어 있어야 함.

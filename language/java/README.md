# Java Language

## Import

### Eclipse

Eclipse -> Import -> Existing Gradle Project

### Intellij

Intellij -> Project -> Open -> Gradle Project

Editor -> Code Style -> Import `./rule/***.xml`

## Run

Type on each classes which contains main method

### Eclipse

- Mac: `Command + F11`
- Windows: `Ctrl + F11`

### Intellij

- `Ctrl + Shift + R`

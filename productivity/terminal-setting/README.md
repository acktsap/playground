# Terminal

## Setup

- Setup : `setup.sh`
- Setup vim : `vi .` and :PluginInstall

## Color

https://github.com/sindresorhus/hyper-snazzy

### Iterm

- Import color presets : Profiles -> colors -> color presets -> import, import Snazzy.itermcolors
- Text size : Profile -> Text, text size as 13
